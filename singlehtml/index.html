Multiedit v 1.1¶

1. Start¶

Assuming that you have already installed ths plugin from Plugin manager (zip file or direct download) click on the plugin icon

in the plugintoolbar or use the menu entry from the vector menu. The first window shown is the main dialog window of the plugin:

A. Modify attributes on a subset of features¶

  1. choose layer from the Layer combobox making sure it is a vector layer ...
  1. choose field from the Field combobox and from the
  1. choose attribute value to modify from the Old attribute value combobox and from the Field to write the value to combobox
  1. click on Select Features button and read the outcome in the info window below
  1. click on the write value to chosen field button and after you may click on show attribute table to check result from attribute table
  1. then either click on save or on exit button to complete operation.
  2. Either if you have chosen to save your edits or not you have to manually stop editing.

B. Modify attributes on all features of a layer¶

  1. choose layer from the layer combobox
  1. choose field from the field combobox
  1. Click on the button Select all
  1. click on the write value to chosen field button and after you may click on show attribute table to check result from attribute table
  2. then either click on save or on exit button to complete operation.
  3. Either if you have chosen to save your edits or not you have to manually stop editing.

C.Create a new field to write your new values to a subset of selected features or to all features¶

  1. choose layer from the layer combobox making sure it is a vector layer type
  1. insert a new field name in the New Field Name combobox
  2. choose field type from the Field type list of values
  3. press the Create New Field button
  1. choose attribute value to modify from the Old attribute value combobox and from the Field to write the value to combobox choosing the newly create field
  1. follow instructions from point B.3 or A.4
